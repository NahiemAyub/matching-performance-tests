Gatling Suite
===========================

Prerequisites
-------------

* Java 8+
* Maven
* Scala 2.11

Getting started
---------------------

To run a specific simulation:
```text
mvn gatling:execute -Dsimulations.GetRequestsLoadTest
```

To run all simulations:
```text
mvn gatling:execute
```
